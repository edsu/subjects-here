Subjects Here

Below is a map of FAST subjects for your current location (10km radius). If you click on one of the markers you should be able to find books about that location from WorldCat. If you own an iPhone or equivalent add this page to your Home Screen and you should be able to discover books about wherever you happen to be.

subjects-here is an HTML5 experiment (all HTML and JavaScript) which uses the geo-location features in modern browsers to look up your location in OCLC's mapFAST webservice, and display the results using Google Maps API
