Subjects Here

Below is a list of FAST subjects for your current location (10km radius). If you click on one of the subjects you should be able to see books about your current location in WorldCat.

Subjects-Here is a pure html5 experiment (all html and javascript) which uses the geo-location features in modern browsers to look up your location in OCLC's mapFAST webservice.

If you own an iPhone or equivalent (and are an unabashed library geek) add this page to your Home Screen, and you should be able to see subjects & discover books about wherever you happen to be.

The code is all HTML and JavaScript sitting in your browser, but if you want to modify it please send me pull requests on GitHub.
